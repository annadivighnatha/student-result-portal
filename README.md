# Student Result Portal

A Flask-based web application to search student results using hall ticket number.

## Features

- Search results using hall ticket number
- SQLite database integration
- Admin panel to add students
- PDF result download
- Responsive UI

## Technologies

- Python
- Flask
- SQLite
- HTML
- CSS
- ReportLab

## Run Project

Clone repository:

git clone repository-url

Install dependencies:

pip install -r requirements.txt

Run:

python app.py

Open:

http://127.0.0.1:5000
